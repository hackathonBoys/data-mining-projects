# Decision Tree Classification

## Objective
Build a decision tree model to classify outcomes and explain decisions using feature splits.

## What This Project Covers
- Data cleaning and preprocessing
- Feature selection and target definition
- Train-test split
- Training a DecisionTreeClassifier
- Tree visualization and feature importance
- Model evaluation using accuracy, confusion matrix, and classification report

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Outcome
Created an interpretable classification model and analyzed key features influencing predictions.
