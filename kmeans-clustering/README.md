# K-Means Clustering

## Objective
Use unsupervised learning to group data into meaningful clusters for pattern discovery and analysis.

## What This Project Covers
- Data cleaning and preprocessing
- Feature scaling using StandardScaler
- Elbow Method to determine optimal number of clusters
- K-Means model training
- Cluster visualization and interpretation

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Outcome
Identified distinct data segments that can be used for business insights, customer segmentation, or exploratory analysis.
